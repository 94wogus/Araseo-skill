# 회원가입 플로우 테스트

사용자가 새 계정을 만드는 과정을 보여주는 플로우차트

## 주요 단계

1. **시작** - 사용자가 회원가입 클릭
2. **정보 입력 화면** - 이름, 이메일, 비밀번호 입력
3. **유효성 검사** - 입력 데이터 검증
4. **이메일 중복 확인** - 이미 사용중인 이메일인지 체크
   - 중복 → 에러 메시지 표시
   - 신규 → 계정 생성
5. **계정 생성** - 데이터베이스에 저장
6. **환영 이메일 발송** - 가입 환영 메일 전송
7. **완료** - 회원가입 성공

## 플로우

```
시작
  ↓
정보 입력 화면
  ↓
유효성 검사
  ↓
이메일 중복 확인?
  ├─ 중복 → 에러 메시지 → 정보 입력 화면
  └─ 신규 → 계정 생성
                ↓
            환영 이메일 발송
                ↓
              완료
```

## 결정 포인트

- **유효성 검사**: 필수 필드 누락 시 → 에러
- **이메일 중복 확인**: 기존 이메일 → 다시 입력 요청

## 색상 가이드

- 시작/완료: 초록색 (성공)
- 입력/처리: 파란색 (프로세스)
- 결정: 주황색 (판단 필요)
