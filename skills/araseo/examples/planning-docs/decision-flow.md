# 문서 승인 프로세스

제출된 문서가 승인 과정을 거치는 워크플로우입니다.

## 프로세스 단계

1. **문서 제출** - 작성자가 문서 제출
2. **1차 검토** - 팀 리더가 초기 검토
3. **검토 결과 판단**
   - 승인 → 다음 단계
   - 반려 → 수정 필요
4. **2차 승인** - 관리자 최종 승인
5. **최종 승인 판단**
   - 승인 → 게시
   - 반려 → 수정 필요
6. **문서 게시** - 최종 승인 시 게시
7. **수정 요청** - 반려 시 작성자에게 수정 요청

## 흐름 다이어그램

```
문서 제출 → 1차 검토 → 검토 결과?
                          ├─ 승인 → 2차 승인 → 최종 승인?
                          │                      ├─ 승인 → 문서 게시
                          │                      └─ 반려 → 수정 요청
                          └─ 반려 → 수정 요청
```

수정 요청 후 → 다시 문서 제출로 돌아감 (순환)
